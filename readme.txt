Se adjunta en la carpeta raíz library la base de datos en el archivo library.sql para ser importada en el 
phpmyadmin y se pueda ejecutar la aplicación web.
Ya dentro del navegador deben ir a la url localhost/library/ en la parte de arriba se encuentra un menu
con vínculos a cada una de las tablas al ejecutar cada uno de estos se despliegan todos las instancias de su
respectiva tabla, junto con un botón para ver la información de la instancia, editarla o eliminarla 
y además se muestra un botón que dirige al vínculo para crear una nueva instancia.
